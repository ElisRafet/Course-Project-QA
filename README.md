# Automated Software Testing – Course Project
**University of Economics – Varna**

## 📝 Project Overview
This repository contains a comprehensive software testing project conducted for the **"Automated Software Testing"** course. We performed manual and automated testing on a food delivery platform, identifying logic and UI bugs.
